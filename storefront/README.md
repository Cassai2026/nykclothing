## ?? Deployment & Operations

This frontend is entirely decoupled from the database architecture. This means it can be deployed on any edge network (like Fasthosts) for 100% uptime, instant global load times, and maximum SEO indexing.

1. **Start the Master Vault:** Ensure the backend Node.js/PostgreSQL server is live and routing traffic.
2. **Launch Storefront:** Deploy index.html to initialize the customer-facing UI.
3. **Live Sync:** The client automatically syncs with the master database to display active products, verify inventory, and route securely to Stripe.
